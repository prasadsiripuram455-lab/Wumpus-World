The 5x5 Grid Environment

An intricate grid where danger and treasure coexist. The agent relies on sensory inputs to make rational decisions.

Find the Gold! Avoid the Wumpus and Pits.

Game Elements

  • Agent (A): Starts at safe cell (0,0), can move up, down, left, right.
  • Gold (G): Ultimate goal. Reaching it ends the game successfully.
  • Wumpus (W): A deadly monster. Encountering it ends the game.
  • Pits (P): Bottomless traps. Falling in equals game over.

Sensory Signals

  • 💨 Breeze: Indicates a Pit is in an adjacent cell.
  • 👃 Smell: Indicates the Wumpus is nearby.
  • ✨ Glitter: Indicates the presence of Gold in the current cell.

The Search Agents

Four students implemented unique optimization and search algorithms to tackle the Wumpus problem.

Breadth First Search (BFS)

O(b^d) Time

Explores nodes level by level using a queue (FIFO). Guarantees the shortest path in an unweighted graph and ensures all nodes at the current depth are verified before advancing. It is a complete algorithm.

Bidirectional Search

O(b^(d/2)) Time

Runs two searches simultaneously: forward from the start and backward from the goal. By meeting in the middle, it drastically reduces the search space and improves efficiency over standard BFS.

Simulated Annealing

Optimization

A probabilistic algorithm inspired by metallurgy (heating and cooling). Initially explores many paths randomly using high "temperature," gracefully converging to a near-optimal solution to escape local optima.

Depth First Search (DFS)

O(b^m) Time

Explores deeply down a single branch before backtracking, utilizing a stack (LIFO) or recursion. High memory efficiency, though it does not guarantee finding the shortest path first.

Greedy Best-First Search

Heuristic Based

Selects the node appearing closest to the goal using a Manhattan distance heuristic. Extremely fast processing speed, but often bypasses the true optimal route by ignoring actual path costs.

Genetic Algorithm

Evolutionary

A population-based approach using selection, crossover, and mutation to iteratively improve random starting solutions over generations. Excellent for highly complex, vast search spaces.

Uniform Cost Search (UCS)

Optimal

Expands nodes strictly by the lowest cumulative path cost using a priority queue. Behaves identically to Dijkstra’s algorithm and guarantees the most optimal path when all step costs are positive.

A* Algorithm

f(n) = g(n) + h(n)

The gold standard for pathfinding. Intelligently combines the known cost to reach a node g(n) with an estimated heuristic cost to the goal h(n). Assures optimal routing and maximal efficiency.

Minimax Algorithm

Adversarial

Primarily used in game theory. Evaluates subsequent moves assuming the "opponent" plays optimally. It seeks to maximize the agent's gain while fiercely minimizing the environment's threat potential.

Iterative Deepening (IDS)

DFS + BFS Hybrid

Executes repeated DFS passes with incrementally expanding depth limits. Retains the extraordinary memory efficiency of DFS whilst matching the optimal, complete guarantee of BFS.

Recursive Best First (RBFS)

Memory-efficient A*

Explores promising nodes strictly guided by A*'s evaluation function f(n), while operating via lightweight recursion and limited tracking memory instead of expanding enormous priority queues.

Constraint Satisfaction (CSP)

Logical Reasoning

Translates sensor triggers (Breeze, Smell) into complex inter-variable logical constraints over a domain of cells. Heavily employs constraint propagation and systematic backtracking search.
